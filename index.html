What is Jappy Framework ?

Jappy Framework makes Java happy to use for many common use cases like working with databases, logging, sending emails or doing some crawl. Java is a beautiful language. Some of the best libraries are stable in Java. Other languages/frameworks are attracting early adopters with simple syntax, quick setup and Java is loosing with its complex config files, factory stuff to do simple things, etc. Jappy-Framework aims to fix that.

We follow the following philosophy

  • No Config Files
  • Simple static API
  • Beautiful/Powerful Admin Interfaces
